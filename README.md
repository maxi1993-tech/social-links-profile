### What I learned

This project helped me consolidate what I learned on the previous one: `@font-face`, CSS custom properties, Flexbox. I also learned how to target a child element on parent hover using `li:hover a`.

This project is still a work in progress. I plan to come back and replace `px` with relative units (`rem`, `em`, `%`), explore `clamp()` for responsive typography, and clean up the CSS (class naming, removing redundant rules).

### Continued development

On future projects I want to keep refining my HTML/CSS habits — semantics, class naming, code organization. I also want to seriously get into relative units (`rem`, `em`, `%`) for better responsive design, and improve my pixel perfect skills.
